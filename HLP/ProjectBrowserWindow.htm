Project Browser Window

The Project Browser window shows all open FasterCap, FastCap2 and FastHenry2 files.

To bring to front any edit window, double-click with the left mouse button on the corresponding window name in the Project Browser.

To delete any FasterCap / FastCap2 input window, select the window name name in the Project Browser and press the Del key (Remark: you can only delete secondary FasterCap / FastCap2 windows, not the main FasterCap / FastCap2 view. To close completly a FasterCap / FastCap2 file, see Close FastCap2 Files).
