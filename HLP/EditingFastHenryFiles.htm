Editing FastHenry2 Files

You can modify your FastHenry2 input files in the FastHenry2 View.

As you type, you will notice that the text will be automatically colored according to the semantic meaning of the command; that is, color highlighting is not performed on a simple token basis, but is fully context-sensitive.

In this way, you will see highlighted in red the instructions that would generate an error when parsed with FastHenry2; therefore you will be able to correct them in advance.

To update the FastModel View according to the changes you make to the input file, right-click the mouse while the cursor is on the FastHenry2 View and select Update Model from the pop-up menu.

To save, select File->Save FastHenry2 File or File->Save FastHenry2 File As from the main menu.
