Importing FasterCap / FastCap2 Files

To import a FasterCap / FastCap2 file, activate the FastCap2 window by clicking on it, then select File->Open FastCap2 File from the main menu (do not select the Open Project) and browse to your file. You will notice that you can import either .lst, .neu or .qui / .txt files; the distinction is the same as in FastCap2 (for details, see FastCap2 manual):

  • The .lst files are list files, that is, they contain references to other files to be hierarchically included
  • The .qui / .txt files are generic-format input files (files containing 'Q' or 'T' definitions
  • The .neu files are Patran input files. Remark: Patran input files are not supported by FasterCap.

Note, however, that though FastModel does not distinguish between .lst and .qui files, FastCap2 does, so it is recommended to work with the default extensions.
On the other hand, the only files recognized as Patran format input files are those whose extension is .neu.
In case of FasterCap, all files can be hierarchical, so there is no distinction between a .lst, .qui or .txt file.

When you have performed your selection, the file(s) will be imported. This could take some time if the input file is large, please be patient. At the end you should see your file in the FastCap2 View, with keywords highlighted in color, and a view of your model in the FastModel View.

Any error occurred while importing will be reported in the Output Window.

If you import a list file, FastModel will open as many windows as the number of different files referenced in the list file. These windows are minimized by default; you can activate them by clicking on the captions (you may need to move the windows in foreground to see them) or selecting them in the Project Browser.

The Project Browser will keep trace of all open FastCap2 files. Note that there will always be a main FastCap2 View, usually corresponding to the list (.lst) file, and any number of secondary FastCap2 windows, usually corresponding to files included by the list file. As you could also generate included files from scratch, you can open new, blank FastCap2 secondary windows by selecting 'New FastCap2 File' from the 'File' menu.
See Project Browser Window for more information on the Project Browser.

Remark: Patran input files can be imported but they cannot be edited inside FastModel.
