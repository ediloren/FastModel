Manuals

You can access the original User's Manuals for FastCap and FastHenry directly from FastCap2 and FastHenry2 on-line help.

Please note that these are the original manuals for the Unix version, so they do not completely apply to the Windows version, even if only in minor aspects.
In particular, the FastCap manual contains references to many scripts to be run under Unix. These scripts have been converted for the DOS shell under Windows. You'll find them bundled together with the sample files.
