Editing FasterCap / FastCap2 Files

You can modify your FastCap2 input files in the FastCap2 View and windows.

As you type, you will notice that the text will be automatically colored according to the semantic meaning of the command; that is, color highlighting is not performed on a simple token basis, but is fully context-sensitive.

In this way, you will see highlighted in red the instructions that would generate an error when parsed with FastCap2; therefore, you will be able to correct them in advance.

To update the FastModel View according to the changes you make to the input file(s), right-click the mouse while the cursor is on any FastCap2 window and select Update Model from the pop-up menu.

To save, select File->Save FastCap2 File or File->Save FastCap2 File As from the main menu.
