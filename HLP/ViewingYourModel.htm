Viewing Your Model

The FastModel View windows allows you to see and move in real time the 2D or 3D representation of your FasterCap/FastCap2/FastHenry2 input files.

Move around your model

As soon as you have imported an input file or have typed in something and performed an Update Model operation, you will be able to view your model in the FastModel View(s), zoomed to fit into the window. Then you can move around your model; to do so, use the left mouse button together with the shift and control keys:

  • to pan the image, click the mouse left button and move the cursor
  • to zoom in and out, hold down the control key while clicking the mouse left button and moving the cursor up and down
  • to rotate the model around the center of the view, hold down the shift key while clicking the mouse left button and moving the cursor
  • rotate the model around you, hold down the shift and control keys together while clicking the mouse left button and moving the cursor

If you get lost while moving (i.e. you cannot see the model any more), click the right button while on the FastModel View and select Zoom All from the pop-up menu.

FastModel View options

You can also select other options from the pop-up menu:

  • under Display you can select to remove or not the hidden lines or to have a solid view of your object
  • under View you can select to have a 2D or 3D view
  • under Autoset 2D you can automatically rotate the model (only when in 2D mode) to show orthographic views
  • under Properties you can set colors for your current View

Add FastModel Views

You can also add as many FastModel Views as you want. Each View is independent from the others, that is, can have different point of view, 2D/3D settings, display mode and colors.
To add new views, select Window->New window from the main menu.

Print, print preview and exporting images

You can also print and print preview your model exactly as you see it in the FastModel View of your choice; just select 'Print' or 'Print Preview' from File drop down menu.
You can also print directly to an Encapsulated PostScript (EPS) file, useful for direct inclusion in your documents: EPS is a vector picture format, so you can zoom or scale the picture without loss of resolution. You can open the EPS file with a PostScript viewer (like the freeware GhostView) or directly import it into your document.
For instance, in MS Word, select Insert->Picture->From File on the main menu to import the picture into a sheet.
