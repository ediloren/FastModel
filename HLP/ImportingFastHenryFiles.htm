Importing FastHenry2 Files

To import a FastHenry2 file, activate the FastHenry2 View by clicking on it, then select File->Open FastHenry2 File from the main menu (do not select Open Project) and browse to your file.

When you have performed your selection, the file will be imported. This could take some time if the input file is large, please be patient. At the end you should see your file in the FastHenry2 View, with keywords highlighted in color, and a view of your model in the FastModel View.

Any error occurred while importing will be reported in the Output Window.
