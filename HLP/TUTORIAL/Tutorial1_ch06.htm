Change the visualization mode

  1. Load the FastHenry2 30pin.inp sample file as described in Load FastHenry2 input files
  2. Bring the FastModel View in foreground

Select between wireframe, hidden lines or solid view

  1. Right-click with the mouse on the FastModel View and on the pop-up menu choose Display->Hidden Lines or select the icon on the toolbar. The model will be redrawn hiding the lines which fall under the model surfaces.
  2. Right-click with the mouse on the FastModel View and on the pop-up menu choose Display->Solid or select the icon on the toolbar. The model will be redrawn as a shadowed solid.
  3. Right-click with the mouse on the FastModel View and on the pop-up menu choose Display->Wireframe or select the icon on the toolbar. The model will be shown as wireframe lines.

Select between 2D or 3D view

  1. Right-click with the mouse on the FastModel View and on the pop-up menu choose View->2D (Orthographic). The model will be redrawn in an orthographic view.
  2. Right-click with the mouse on the FastModel View and on the pop-up menu choose Autoset->XY Plane. The model X, Y axes will be adjusted perpendicular to the view (you may need to pan the model in the view).
  3. Right-click with the mouse on the FastModel View and on the pop-up menu choose Autoset->YZ Plane. The model Y, Z axes will be adjusted perpendicular to the view (you may need to pan the model in the view).
  4. Right-click with the mouse on the FastModel View and on the pop-up menu choose Autoset->ZX Plane. The model Z, X axes will be adjusted perpendicular to the view (you may need to pan the model in the view).
  5. Right-click with the mouse on the FastModel View and on the pop-up menu choose View->3D (Perspective). The model will be redrawn in perspective.
