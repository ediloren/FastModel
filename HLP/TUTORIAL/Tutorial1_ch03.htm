Load FastHenry2 input files

To load a FastHenry2 input file:
  1. Select the FastHenry2 View (the view should be in foreground with the caption highlighted).
  2. From the File menu select File->Open FastHenry2 File or click on the icon on the toolbar.
  3. Browse to the 'Shared Documents' folder (in the English version of Windows XP, this usually is 'C:\Documents and Settings\All Users\Shared Documents', or in Windows Vista and Windows 7, this usually is the folder 'Public') and under FastFieldSolvers\FastHenry2 select the file 30pin.inp, then click Open.

The cursor will change to a glasshour until the file is imported into FastModel. As soon as the import finishes, you should be able to see the 3D model of the 30-pin connector in the FastModel View.

You will also notice that the FastHenry2 View now contains the input file and that the file has been colored according to the input semantic. You can type in the view and see the colors change according to what you write.

  1. Select the FastHenry2 View and from the File menu choose File->Close FastHenry2 File to close the file.
