FastModel Tutorial

Getting started with FastModel

This tutorial will show you step-by-step how to perform the basic visualization tasks with FastModel.

In this tutorial, you will learn how to:

  • Run FastModel and navigate its panels
  • Load FastHenry2 input files
  • Load FastCap2 input files
  • Zoom, rotate and pan the model
  • Change the visualization mode
  • Perform changes in the input file and update the view
  • Change the color preferences
  • Print and export the current view
  • Work with multiple FastModel views
