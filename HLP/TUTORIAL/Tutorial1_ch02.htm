Run FastModel and navigate its panels

Launch FastModel

  1. If FastModel is not already running, double click the FastModel icon on your desktop or select 'FastModel' from your Start menu in Windows.
  2. Click on the I agree button to accept FastModel's license.

FastModel View

The FastModel View is the main window where the model is shown and can be manipulated. It usually has a gradient background (but you can apply your preferences). The View should be empty since we have not loaded any model yet.
  1. Move the around view and resize it according to your preferences

FastCap2 View

The FastCap2 View shows the textual input file for FastCap2. If you cannot see the view, perform the following steps:
  1. On the View menu, select View->FastCap2 View.
    2. You can try writing in this window. Probably, you'll notice that what you type is marked in red. This is because FastModel is syntax-checking the text and marks unknown statements as errors.
  2. Move around the view and resize it according to your preferences.

FastHenry2 View

The FastHenry2 View shows the textual input file for FastHenry2. If you cannot see the view, perform the following steps:
  1. On the View menu, select View->FastHenry2 View.
You can try writing in this window. Probably you'll notice that what you type is marked in red. This is because FastModel is syntax-checking the text and marks unknown statements as errors.

Browser Window

The Browser Window shows the hierarchy of the input files loaded into FastModel. If you cannot see the window, perform the following steps:
  1. On the View menu, select View->Browser Window.
This window can be left floating, docked to the sides of the frame window, moved and resized. If you double click on the FastHenry2 or FastCap2 icons, the FastHenry2 and FastCap2 views will pop up.

The Browser Window will show all the input files that you have loaded into FastModel (at the moment none).

Output Window

The Output Window shows any message FastModel will issue during file import or update. These messages are usually warning or error messages related to the input files.

You can test the Output window in the following way:

  1. Bring to front the FastCap2 View, either selecting View->FastCap2 View or by double clicking on the FastCap2 icon in the Browser Window
  2. Write something in the FastCap2 View
  3. Right click the mouse with the cursor on the FastCap2 View area and select 'Update Model' on the pop-up menu
  4. Examine the Output Window: it should contain some error messages
