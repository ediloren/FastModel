Work with multiple FastModel views

  1. Load the FastHenry2 pin-connect.inp sample file as described in Load FastHenry2 input files.
  2. Bring the FastModel View in foreground, move and rotate it according to your preferences to have a full view of the model.

Create the main view

  1. In the FastModel View, select a 3D solid view.
  2. Change the FastModel View colors.
  3. Resize the FastModel View to occupy the upper left quarter of your canvas.

Create the auxiliary views

  1. On the main menu choose Window->New Window. A new FastModel View window will appear.
  2. Resize the FastModel View to occupy the upper right quarter of your canvas
  3. Select a 2D wireframe view for this FastModel View and autoset it to the YZ plane
  4. On the main menu choose Window->New Window. A new FastModel View window will appear.
  5. Resize the FastModel View to occupy the lower right quarter of your canvas
  6. Select a 2D wireframe view for this FastModel View and autoset it to the XY plane
  7. On the main menu choose Window->New Window. A new FastModel View window will appear.
  8. Resize the FastModel View to occupy the lower left quarter of your canvas
  9. Select a 2D wireframe view for this FastModel View and autoset it to the ZX plane

Verify the persistence of your settings

  1. Close FastModel
  2. Lauch FastModel again

You'll notice that the windows and views positions and colors have been preserved.
