Load FastCap2 input files

To load a FastCap2 input file:
  1. Select the FastCap2 View (the view should be in foreground with the caption highlighted).
  2. From the File menu select File->Open FastCap2 File or click on the icon on the toolbar.
  3. Browse to the 'Shared Documents' folder (in the English version of Windows XP, this usually is 'C:\Documents and Settings\All Users\Shared Documents', or in Windows Vista and Windows 7, this usually is the folder 'Public') and under FastFieldSolvers\FastCap2 select the file coated_sph.lst, then click Open.

The cursor will change to a glasshour until the file is imported into FastModel. As soon as the import finishes, you should be able to see the 3D model of the coated sphere in the FastModel View.

You will also notice that the FastCap2 View now contains the input file and that the file has been colored according to the input semantic. You can type in the view and see the colors change according to what you write.

Moreover, in the Browser Window you should see near the FastCap2 icon a '+' mark, showing that more than one file has been loaded.

  1. Click on the '+' mark near the FastCap2 icon in the Browser Window to expand the list.
  2. Double click on the name 'sphere2.qui'. You will see another window appear in foreground.

This is because FastCap2 files, unlike FastHenry2 ones, can have a 2-level hyerarchy. A main file can refer to any number of subfiles. This eases the task to manage big models and provide a simple way to specify dielectric interfaces, as you will see in the FastCap2 tutorials.

  1. Select the FastCap2 View and from the File menu choose File->Close FastCap2 File to close the file.
