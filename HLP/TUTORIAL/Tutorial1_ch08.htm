Change the color preferences

  1. Load the FastHenry2 pin-connect.inp sample file as described in Load FastHenry2 input files.
  2. Bring the FastModel View in foreground, move and rotate it according to your preferences to have a full view of model.

Change the colors

  1. Select the FastModel View and on the main menu choose View->Properties, or right-click with the mouse on the FastModel View and on the pop-up menu choose Properties. The preferences window will appear.

  2. Click on any colored button to select a new color for any of the described items.
  3. Press Ok to close the window and apply the changes.
