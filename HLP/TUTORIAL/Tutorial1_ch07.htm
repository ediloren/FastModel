Perform changes in the input file and update the view

  1. Load the FastHenry2 vias.inp sample file as described in Load FastHenry2 input files.
  2. Bring the FastModel View in foreground, move and rotate it according to your preferences to have a full view of the meshed plane and the vias.

Change the input file

  1. In the FastHenry View, comment out with a '*' the lines belonging to the meshed plane, as shown in the following figure.

  2. Right-click with the mouse on the FastHenry2 View and on the pop-up menu choose Update Model. The model will be redrawn without the meshed plane, thus reflecting the changes you applied to the input file.
  3. In the FastHenry2 View, uncomment the lines you previously commented out, by removing the '*'.
  4. Right-click with the mouse on the FastHenry2 View and on the pop-up menu choose Update Model. The meshed plane will reappared in the FastModel view.
