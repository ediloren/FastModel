Print and export the current view

  1. Load the FastHenry2 pin-connect.inp sample file as described in Load FastHenry2 input files.
  2. Bring the FastModel View in foreground, move and rotate it according to your preferences to have a full view of model.

Print

  1. Select the FastModel View and on the main menu choose File->Print Preview. The preview window will appear.
  2. Press the upper left button Print to actually send the drawing to the printer.

Export in Encapsulated PostScript format

  1. Select the FastModel View and on the main menu choose File->Print to EPS. A file save window will appear.
  2. Select a directory and a file name and save your file.
  3. You can now open your file with a PostScript viewer (like the freeware GhostView) or directly import it into your document. For instance, in MS Word, select Insert->Picture->From File on the main menu to import the picture into a sheet.
