FastModel User Guide

Welcome to FastModel !

FastModel is a powerful text-based 3D editor that allows you to see in real time the volumes you describe in your FasterCap/FastCap2/FastHenry2 input file. Features include color keywords highlighting, unlimited number of 3D views of the model, direct interface to the Solvers, and many others!

Features include:

  • FasterCap and FastCap2 input file support, including Patran interface
  • FastHenry2 input file support
  • Color semantic highlighting (highlighting is not performed on token basis but is context-sensitive)
  • Unlimited number of 3D views of the model
  • Unlimited number of 2D views of the model (orthogonal included)
  • FasterCap, FastCap2 and FastHenry2 integration (you can launch FasterCap, FastCap2 or FastHenry2 on your input file with a simple click)
  • Model View print and print to file in Encapsulated PostScript format (to easily include the model views in you documents)!
  • ..and many others!
