FastModel is a front- and back- end text modeling and visualization software for
FastCap2 and FastHenry2, and to some extent for FasterCap 

For pre-compiled binaries, support, consultancy and additional information please visit http://www.fastfieldsolvers.com/
Access to the download pages is free, and you may access anonymously if you want.

---------------------------
How-to-compile instructions
---------------------------

FasterCap can be compiled for MS Windows using Visual C++ 6.0 or compatible version.

Required dependencies are: 

- HTML Help workshop, from MicroSoft, to compile the .chm embedded online help
